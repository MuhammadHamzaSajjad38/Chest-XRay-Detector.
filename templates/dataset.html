Dataset Details

This project was trained on a custom dataset created by combining several publicly available chest X-ray collections from Kaggle. The data was carefully balanced using augmentation and oversampling techniques to ensure the model does not become biased towards any single class.

Primary Source Example: COVID-19 Radiography Database on Kaggle

Class Summary

The final dataset was organized into four distinct classes for our multi-class classification task. The table below shows the total number of images used in our final dataset after balancing and splitting.
